# 🪟 Script de Compilación para Windows (PowerShell)\n\n## 📱 compile-windows.ps1\n\nScript profesional de compilación para Windows PowerShell que automatiza completamente el proceso de compilación de ambas versiones del Gestor de Créditos (Admin e Invitado).\n\n## ✨ Características\n\n### 🎯 Compilación Automatizada\n- ✅ Ambas versiones en un comando\n- ✅ Verificación automática de requisitos\n- ✅ Instalación automática de dependencias\n- ✅ Gestión robusta de errores\n- ✅ Logging detallado\n\n### 📦 Versiones Múltiples\n- ✅ Versión Admin (permisos completos)\n- ✅ Versión Invitado (solo lectura)\n- ✅ Compilación independiente de cada versión\n- ✅ Configuración automática por versión\n\n### 🚀 Características Avanzadas\n- ✅ Compilación paralela (experimental)\n- ✅ Instalación automática en dispositivo\n- ✅ Generación de reportes detallados\n- ✅ Salida con colores y emojis\n- ✅ Validación de parámetros\n\n### 🔧 Profesional\n- ✅ Manejo robusto de errores\n- ✅ Tiempos estimados\n- ✅ Documentación completa\n- ✅ Solución de problemas\n- ✅ Soporte multiplataforma\n\n## 🚀 Inicio Rápido\n\n### Paso 1: Permitir Ejecución de Scripts\n\nAbre PowerShell como administrador y ejecuta:\n\n```powershell\nSet-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser\n```\n\n### Paso 2: Compilar\n\n```powershell\n.\\compile-windows.ps1\n```\n\n### Paso 3: Esperar\n\nTiempo estimado: **30-45 minutos**\n\n### Paso 4: Instalar (Opcional)\n\n```powershell\n.\\compile-windows.ps1 -Action install\n```\n\n## 📖 Uso\n\n### Sintaxis Básica\n\n```powershell\n.\\compile-windows.ps1 [OPCIONES]\n```\n\n### Opciones\n\n```powershell\n-Action [acción]     # Acción a realizar (predeterminado: all)\n-Parallel            # Compilar en paralelo (experimental)\n-Verbose             # Mostrar salida detallada\n-Help                # Mostrar ayuda\n```\n\n### Acciones\n\n| Acción | Descripción | Tiempo |\n|--------|-------------|--------|\n| `all` | Ambas versiones | 30-45 min |\n| `admin` | Solo Admin | 15-20 min |\n| `guest` | Solo Invitado | 15-20 min |\n| `gradle` | Con Gradle local | 20-30 min |\n| `install` | Instalar en dispositivo | 1-2 min |\n| `clean` | Limpiar archivos | < 1 min |\n| `report` | Generar reporte | < 1 min |\n| `help` | Mostrar ayuda | Inmediato |\n\n## 💡 Ejemplos\n\n### Compilar Ambas Versiones\n\n```powershell\n.\\compile-windows.ps1\n```\n\n### Compilar Solo Admin\n\n```powershell\n.\\compile-windows.ps1 -Action admin\n```\n\n### Compilar Solo Invitado\n\n```powershell\n.\\compile-windows.ps1 -Action guest\n```\n\n### Compilar en Paralelo\n\n```powershell\n.\\compile-windows.ps1 -Action all -Parallel\n```\n\n### Modo Detallado\n\n```powershell\n.\\compile-windows.ps1 -Verbose\n```\n\n### Instalar en Dispositivo\n\n```powershell\n.\\compile-windows.ps1 -Action install\n```\n\n### Mostrar Ayuda\n\n```powershell\n.\\compile-windows.ps1 -Action help\n```\n\n## 📋 Requisitos\n\n### Obligatorios\n\n- **PowerShell 5.0+**\n  - Incluido en Windows 10+\n  - Verificar: `$PSVersionTable.PSVersion`\n\n- **Node.js 18+**\n  - Descargar: https://nodejs.org\n  - Verificar: `node --version`\n\n- **npm 8+**\n  - Incluido con Node.js\n  - Verificar: `npm --version`\n\n- **EAS CLI**\n  - Instalar: `npm install -g eas-cli`\n  - Verificar: `eas --version`\n\n### Opcionales\n\n- **Android SDK** (para compilación con Gradle)\n- **adb** (para instalar en dispositivo)\n\n## 🔧 Instalación de Requisitos\n\n### Opción 1: Script Automatizado\n\n```powershell\n.\\setup-dependencies.ps1\n```\n\n### Opción 2: Manual\n\n**Node.js:**\n1.
