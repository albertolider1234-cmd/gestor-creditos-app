# 🪟 Gestor de Créditos y Cobranzas - Windows Edition\n\n## 📱 Aplicación Android Profesional\n\nAplicación completa de gestión de créditos y cobranzas para Android con dos versiones:\n- **Admin**: Acceso completo\n- **Invitado**: Solo lectura\n\n## 🚀 Inicio Rápido (Windows)\n\n### Requisitos\n\n- **Windows 10+**\n- **PowerShell 5.0+** (incluido)\n- **Node.js 18+** - Descargar de https://nodejs.org\n- **npm 8+** - Incluido con Node.js\n- **EAS CLI** - Se instala automáticamente\n\n### Paso 1: Permitir Ejecución de Scripts\n\nAbre PowerShell como administrador y ejecuta:\n\n```powershell\nSet-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser\n```\n\n### Paso 2: Compilar Ambas Versiones\n\n```powershell\n.\\compile-windows.ps1\n```\n\n**Tiempo estimado:** 30-45 minutos\n\n### Paso 3: Instalar en Dispositivo\n\nConecta tu dispositivo Android y ejecuta:\n\n```powershell\n.\\compile-windows.ps1 -Action install\n```\n\n## 📖 Acciones del Script\n\n| Acción | Descripción | Tiempo |\n|--------|-------------|--------|\n| (sin parámetro) | Compilar ambas versiones | 30-45 min |\n| `-Action admin` | Solo versión Admin | 15-20 min |\n| `-Action guest` | Solo versión Invitado | 15-20 min |\n| `-Action install` | Instalar en dispositivo | 1-2 min |\n| `-Action clean` | Limpiar archivos | < 1 min |\n| `-Action report` | Generar reporte | < 1 min |\n| `-Action help` | Mostrar ayuda | Inmediato |\n\n## 💡 Ejemplos de Uso\n\n### Compilar Ambas Versiones\n```powershell\n.\\compile-windows.ps1\n```\n\n### Solo Versión Admin\n```powershell\n.\\compile-windows.ps1 -Action admin\n```\n\n### Solo Versión Invitado\n```powershell\n.\\compile-windows.ps1 -Action guest\n```\n\n### Compilación Paralela (Más Rápida)\n```powershell\n.\\compile-windows.ps1 -Action all -Parallel\n```\n\n### Modo Detallado\n```powershell\n.\\compile-windows.ps1 -Verbose\n```\n\n### Instalar en Dispositivo\n```powershell\n.\\compile-windows.ps1 -Action install\n```\n\n## 📁 Archivos Generados\n\nDespués de compilar, encontrarás:\n\n```\n.\\dist\\apks\\\n├── gestor-creditos-admin-2026-06-27_14-30-45.apk\n└── gestor-creditos-guest-2026-06-27_14-30-45.apk\n```\n\n## 📊 Características de la Aplicación\n\n### Versión Admin\n- ✅ Gestión completa de deudores\n- ✅ Creación y edición de pagos\n- ✅ Reportes y gráficos\n- ✅ Gestión de usuarios\n- ✅ Notificaciones configurables\n- ✅ Exportación de datos\n\n### Versión Invitado\n- ✅ Visualización de deudores\n- ✅ Consulta de pagos\n- ✅ Visualización de reportes\n- ✅ Solo lectura (sin edición)\n- ✅ Acceso limitado\n\n## 🆘 Solución de Problemas\n\n### Error: \"No se puede ejecutar el script\"\n\n```powershell\nSet-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser\n```\n\n### Error: \"Node.js no encontrado\"\n\n1. Instala Node.js desde https://nodejs.org\n2. Reinicia PowerShell\n3. Verifica: `node --version`\n\n### Error: \"npm install falló\"\n\n```powershell\nnpm cache clean --force\nnpm install\n```\n\n### Error: \"Compilación falló\"\n\n1. Revisa: `.\\dist\\build.log`\n2. Ejecuta: `.\\compile-windows.ps1 -Verbose`\n3. Intenta: `.\\compile-windows.ps1 -Action clean`\n4. Reintenta compilación\n\n## 📚 Documentación Completa\n\n- **COMPILE_WINDOWS_README.md** - Guía rápida\n- **dist/COMPILE_WINDOWS_GUIDE.md** - Guía detallada\n- **dist/GUEST_VERSION_INFO.md** - Info de versión invitado\n- **dist/BUILD_BOTH_VERSIONS.md** - Compilación dual\n\n## 🔐 Seguridad\n\n- Credenciales de EAS guardadas en `%USERPROFILE%\\.eas`\n- No compartir este directorio\n- Script de código abierto\n- Sin modificaciones del sistema\n\n## 🎯 Flujo de Trabajo Recomendado\n\n### Primera Vez\n\n```powershell\n# 1.
