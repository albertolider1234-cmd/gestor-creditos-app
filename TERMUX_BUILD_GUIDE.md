# 📱 Guía de Compilación en Termux (Android)\n\n## 🎯 Objetivo\n\nCompilar la aplicación Gestor de Créditos directamente en tu dispositivo Android usando Termux, sin necesidad de una computadora Windows.\n\n## ⏱️ Tiempo Total: 45-60 minutos\n\n## 📋 Requisitos\n\n### Hardware\n- ✅ Dispositivo Android 8.0+\n- ✅ Mínimo 4 GB de RAM\n- ✅ Mínimo 10 GB de espacio libre\n- ✅ Conexión a Internet estable\n\n### Software\n- ✅ Termux (aplicación)\n- ✅ Gestor de archivos (para transferir archivos)\n- ✅ Tu proyecto (archivos fuente)\n\n## 🚀 Instalación Paso a Paso\n\n### Paso 1: Instalar Termux (5 minutos)\n\n1. Abre Google Play Store en tu dispositivo Android\n2. Busca \"Termux\"\n3. Instala la aplicación oficial de Termux\n4. Abre Termux\n\n### Paso 2: Actualizar Termux (10 minutos)\n\nEn Termux, ejecuta:\n\n```bash\npkg update\npkg upgrade -y\n```\n\n### Paso 3: Instalar Dependencias (15 minutos)\n\nEn Termux, ejecuta:\n\n```bash\npkg install -y nodejs npm git\n```\n\nVerifica que se instalaron correctamente:\n\n```bash\nnode --version\nnpm --version\ngit --version\n```\n\n### Paso 4: Transferir Proyecto (5 minutos)\n\n**Opción A: Usando Git**\n\nSi tu proyecto está en GitHub:\n\n```bash\ncd $HOME\ngit clone https://github.com/tu-usuario/gestor-creditos-app.git\ncd gestor-creditos-app\n```\n\n**Opción B: Usando Transferencia Manual**\n\n1. En tu computadora, comprime la carpeta del proyecto\n2. Transfiere el archivo ZIP a tu dispositivo Android\n3. En Termux, extrae el archivo:\n\n```bash\ncd $HOME/storage/downloads\nunzip gestor-creditos-app.zip\ncd gestor-creditos-app\n```\n\n**Opción C: Usando Almacenamiento Compartido**\n\n```bash\ntermux-setup-storage\ncd $HOME/storage/downloads\n# Copia tu proyecto aquí\ncd gestor-creditos-app\n```\n\n### Paso 5: Compilar APKs (30-40 minutos)\n\nEn Termux, en el directorio del proyecto:\n\n```bash\n# Hacer el script ejecutable\nchmod +x compile-termux.sh\n\n# Compilar ambas versiones\n./compile-termux.sh\n```\n\n**O compilar solo una versión:**\n\n```bash\n# Solo Admin\n./compile-termux.sh admin\n\n# Solo Invitado\n./compile-termux.sh guest\n```\n\n### Paso 6: Encontrar los APKs (2 minutos)\n\nLos APKs estarán en:\n\n```\n$HOME/gestor-creditos-app/dist/apks/\n```\n\nPara listar los archivos:\n\n```bash\nls -lh dist/apks/\n```\n\n### Paso 7: Instalar APKs (5 minutos)\n\n**Opción A: Instalar Directamente desde Termux**\n\n```bash\n# Instalar Admin\nadb install dist/apks/gestor-creditos-admin-*.apk\n\n# Instalar Invitado\nadb install dist/apks/gestor-creditos-guest-*.apk\n```\n\n**Opción B: Transferir y Instalar Manualmente**\n\n1. Abre el administrador de archivos\n2. Navega a: `/data/data/com.termux/files/home/gestor-creditos-app/dist/apks/`\n3. Copia los archivos `.apk` a tu carpeta de Descargas\n4. Abre cada APK para instalar\n5. Permite la instalación desde fuentes desconocidas si es necesario\n\n## 📖 Comandos Disponibles\n\n### Compilar Ambas Versiones\n```bash\n./compile-termux.sh\n```\n\n### Compilar Solo Admin\n```bash\n./compile-termux.sh admin\n```\n\n### Compilar Solo Invitado\n```bash\n./compile-termux.sh guest\n```\n\n### Limpiar Archivos Temporales\n```bash\n./compile-termux.sh clean\n```\n\n### Mostrar Ayuda\n```bash\n./compile-termux.sh help\n```\n\n## 🔍 Monitorear Compilación\n\n### Ver Logs en Tiempo Real\n\n```bash\n# Ver últimas líneas del log\ntail -f dist/build-termux.log\n\n# Ver todo el log\ncat dist/build-termux.log\n```\n\n### Ver Reporte de Compilación\n\n```bash\n# Listar reportes\nls -lh dist/build-report-termux-*.txt\n\n# Ver reporte\ncat dist/build-report-termux-*.txt\n```\n\n## 🆘 Solución de Problemas\n\n### Error: \"Comando no encontrado\"\n\n```bash\n# Asegúrate de estar en el directorio correcto\ncd $HOME/gestor-creditos-app\n\n# Verifica que el script es ejecutable\nls -la compile-termux.sh\n\n# Si no es ejecutable, hazlo ejecutable\nchmod +x compile-termux.sh\n```\n\n### Error: \"Node.js no encontrado\"\n\n```bash\n# Instala Node.js\npkg install -y nodejs npm\n\n# Verifica\nnode --version\nnpm --version\n```\n\n### Error: \"npm install falló\"\n\n```bash\n# Limpia caché\nnpm cache clean --force\n\n# Intenta de nuevo\nnpm install\n```\n\n### Error: \"Espacio insuficiente\"\n\n```bash\n# Verifica espacio disponible\ndf -h\n\n# Limpia caché de Termux\npkg clean\n\n# Limpia caché de npm\nnpm cache clean --force\n```\n\n### Error: \"Compilación falló\"\n\n```bash\n# Revisa el log\ncat dist/build-termux.log\n\n# Intenta limpiar y recompilar\n./compile-termux.sh clean\n./compile-termux.sh\n```\n\n### Error: \"EAS CLI no encontrado\"\n\n```bash\n# Instala EAS CLI globalmente\nnpm install -g eas-cli\n\n# Verifica\neas --version\n```\n\n## 📊 Tiempos Estimados\n\n| Paso | Tiempo |\n|------|--------|\n| Instalar Termux | 5 min |\n| Actualizar Termux | 10 min |\n| Instalar Dependencias | 15 min |\n| Transferir Proyecto | 5 min |\n| Compilar APKs | 30-40 min |\n| Instalar en Dispositivo | 5 min |\n| **Total** | **70-80 min** |\n\n## 🎯 Flujo de Trabajo Completo\n\n```bash\n# 1. Actualizar Termux\npkg update && pkg upgrade -y\n\n# 2. Instalar dependencias\npkg install -y nodejs npm git\n\n# 3. Transferir proyecto (elegir una opción)\n# Opción A: Git\ngit clone https://github.com/tu-usuario/gestor-creditos-app.git\ncd gestor-creditos-app\n\n# 4. Compilar\nchmod +x compile-termux.sh\n./compile-termux.sh\n\n# 5. Esperar 30-40 minutos\n\n# 6. Verificar APKs\nls -lh dist/apks/\n\n# 7. Instalar\nadb install dist/apks/gestor-creditos-admin-*.apk\nadb install dist/apks/gestor-creditos-guest-*.apk\n```\n\n## 📁 Estructura de Directorios\n\n```\n$HOME/\n├── gestor-creditos-app/\n│   ├── src/                    (Código fuente)\n│   ├── dist/\n│   │   ├── apks/               (APKs compilados)\n│   │   ├── build-termux.log    (Log de compilación)\n│   │   └── build-report-*.txt  (Reporte)\n│   ├── compile-termux.sh       (Script de compilación)\n│   ├── app.json                (Config Admin)\n│   ├── app-guest.json          (Config Invitado)\n│   └── package.json            (Dependencias)\n└── storage/\n    └── downloads/              (Descargas)\n```\n\n## 🔐 Seguridad\n\n### Almacenamiento de Credenciales\n\nLas credenciales de EAS se guardan en:\n\n```\n$HOME/.eas\n```\n\nNo compartir este directorio.\n\n### Permisos\n\n- El script solo instala paquetes necesarios\n- No modifica configuración del sistema\n- No descarga código malicioso\n- Script de código abierto\n\n## 💡 Consejos\n\n### Compilación Más Rápida\n\n1. Usa WiFi en lugar de datos móviles\n2. Cierra otras aplicaciones\n3. Mantén el dispositivo conectado a corriente\n4. Usa compilación paralela (si es posible)\n\n### Solucionar Problemas de Memoria\n\n```bash\n# Libera memoria\nfree -h\n\n# Limpia caché\npkg clean\nnpm cache clean --force\n```\n\n### Actualizar Proyecto\n\n```bash\n# Si usas Git\ngit pull origin main\n\n# Recompila\n./compile-termux.sh\n```\n\n## 📞 Soporte\n\n- **Email:** support@gestorcreditos.com\n- **Documentación:** Consulta los archivos .md\n- **Logs:** `dist/build-termux.log`\n- **Reporte:** `dist/build-report-termux-*.txt`\n\n## 🎉 Próximos Pasos\n\n1.
