# 🚀 Guía Rápida de Instalación - Gestor de Créditos Admin\n\n## 📱 Instalación en Dispositivo Android\n\n### Método 1: Usando ADB (Recomendado)\n\n**Requisitos:**\n- Android Debug Bridge (ADB) instalado\n- Dispositivo Android conectado por USB\n- Depuración USB habilitada\n\n**Pasos:**\n\n1. **Conecta tu dispositivo por USB**\n   ```bash\n   # Verifica que el dispositivo esté conectado\n   adb devices\n   ```\n\n2. **Instala el APK**\n   ```bash\n   adb install gestor-creditos-admin.apk\n   ```\n\n3. **Espera a que termine la instalación**\n   ```\n   Success\n   ```\n\n4. **Abre la aplicación**\n   - Busca \"Gestor de Créditos - Admin\" en tu dispositivo\n   - Toca el icono para abrir\n\n### Método 2: Transferencia Directa\n\n1. **Copia el APK a tu dispositivo**\n   ```bash\n   adb push gestor-creditos-admin.apk /sdcard/Download/\n   ```\n\n2. **Abre el administrador de archivos en tu dispositivo**\n   - Ve a Descargas\n   - Busca `gestor-creditos-admin.apk`\n   - Toca para instalar\n\n3. **Permite la instalación desde fuentes desconocidas**\n   - Si se solicita, ve a Configuración → Seguridad\n   - Habilita \"Fuentes desconocidas\"\n   - Intenta instalar nuevamente\n\n### Método 3: Compartir Archivo\n\n1. **Sube el APK a la nube**\n   - Google Drive\n   - Dropbox\n   - OneDrive\n\n2. **Comparte el enlace**\n   - Envía el enlace al usuario\n   - El usuario descarga el archivo\n   - Abre con el instalador de aplicaciones\n\n## 🔧 Habilitar Depuración USB\n\n### Android 6.0 - 9.0\n\n1. Ve a **Configuración**\n2. Desplázate hasta **Acerca del teléfono**\n3. Toca **Número de compilación** 7 veces\n4. Verás el mensaje \"Eres desarrollador\"\n5. Vuelve atrás\n6. Ve a **Opciones de desarrollador**\n7. Habilita **Depuración USB**\n\n### Android 10+\n\n1. Ve a **Configuración**\n2. Toca **Acerca del teléfono**\n3. Toca **Información de compilación** 7 veces\n4. Verás \"Eres desarrollador\"\n5. Ve a **Sistema** → **Opciones de desarrollador**\n6. Habilita **Depuración USB**\n\n## 🎯 Primer Inicio\n\n### Crear Cuenta\n\n1. Abre la aplicación\n2. Toca **Registrarse**\n3. Completa los datos:\n   - Email\n   - Contraseña (mínimo 8 caracteres)\n   - Nombre completo\n4. Toca **Registrarse**\n\n### Iniciar Sesión\n\n1. Abre la aplicación\n2. Ingresa tu email\n3. Ingresa tu contraseña\n4. Toca **Iniciar Sesión**\n\n### Crear Workspace\n\n1. Después de iniciar sesión\n2. Toca **Crear Workspace**\n3. Ingresa el nombre (ej: \"Mi Empresa\")\n4. Toca **Crear**\n\n## 📊 Primeros Pasos\n\n### 1. Agregar Deudor\n\n1. Ve a la pestaña **Deudores**\n2. Toca el botón **+** (flotante)\n3. Completa los datos:\n   - Nombre\n   - Email\n   - Teléfono\n   - Dirección\n4. Toca **Guardar**\n\n### 2. Crear Pago\n\n1. Ve a la pestaña **Pagos**\n2. Toca el botón **+**\n3. Selecciona el deudor\n4. Ingresa:\n   - Monto\n   - Fecha de vencimiento\n   - Tasa de interés\n5. Toca **Guardar**\n\n### 3. Ver Reportes\n\n1. Ve a la pestaña **Reportes**\n2. Selecciona el período (Mes, Trimestre, Año)\n3. Visualiza:\n   - Gráficos de tendencias\n   - Distribución de deuda\n   - Tasa de pago\n4. Exporta a PDF o CSV\n\n### 4. Configurar Notificaciones\n\n1. Ve a la pestaña **Configuración**\n2. Toca **Notificaciones**\n3. Habilita:\n   - Notificaciones generales\n   - Email\n   - WhatsApp\n   - SMS (opcional)\n4. Toca **Guardar**\n\n## ⚙️ Configuración Recomendada\n\n### Seguridad\n\n- ✅ Usar contraseña fuerte (mayúsculas, números, símbolos)\n- ✅ No compartir credenciales\n- ✅ Cerrar sesión en dispositivos públicos\n- ✅ Mantener la app actualizada\n\n### Notificaciones\n\n- ✅ Habilitar alertas de vencimiento\n- ✅ Configurar horario de notificaciones\n- ✅ Habilitar WhatsApp para contacto rápido\n- ✅ Verificar email regularmente\n\n### Datos\n\n- ✅ Hacer backup regularmente\n- ✅ Revisar reportes semanalmente\n- ✅ Exportar datos importantes\n- ✅ Mantener información actualizada\n\n## 🆘 Solución de Problemas\n\n### La aplicación no se instala\n\n**Problema:** \"No se puede instalar la aplicación\"\n\n**Soluciones:**\n1. Verifica que tu dispositivo sea compatible (Android 6.0+)\n2. Desinstala versiones anteriores primero\n3. Libera espacio en tu dispositivo\n4. Intenta instalar desde el administrador de archivos\n\n### La aplicación se cierra\n\n**Problema:** \"La aplicación se detiene inesperadamente\"\n\n**Soluciones:**\n1. Reinicia tu dispositivo\n2. Limpia el caché: Configuración → Aplicaciones → Gestor de Créditos → Almacenamiento → Limpiar caché\n3. Desinstala y reinstala la aplicación\n4. Verifica tu conexión a Internet\n\n### No puedo iniciar sesión\n\n**Problema:** \"Email o contraseña incorrectos\"\n\n**Soluciones:**\n1. Verifica que escribas correctamente tu email\n2. Verifica que escribas correctamente tu contraseña\n3. Usa la opción \"Olvidé mi contraseña\"\n4. Verifica tu conexión a Internet\n5. Contacta al soporte\n\n### Los datos no se sincronizaban\n\n**Problema:** \"Los cambios no aparecen\"\n\n**Soluciones:**\n1. Verifica tu conexión a Internet\n2. Cierra y abre la aplicación nuevamente\n3. Desliza hacia abajo para actualizar (pull-to-refresh)\n4. Reinicia tu dispositivo\n5. Contacta al soporte\n\n### La aplicación es lenta\n\n**Problema:** \"La aplicación responde lentamente\"\n\n**Soluciones:**\n1. Cierra otras aplicaciones\n2. Libera espacio en tu dispositivo\n3. Limpia el caché de la aplicación\n4. Verifica tu conexión a Internet\n5. Reinicia tu dispositivo\n\n## 📞 Soporte\n\n### Contacto\n\n- **Email:** support@gestorcreditos.com\n- **Teléfono:** +1 (555) 123-4567\n- **WhatsApp:** +1 (555) 123-4567\n- **Sitio Web:** https://gestorcreditos.com\n\n### Recursos\n\n- **Documentación:** https://docs.gestorcreditos.com\n- **FAQ:** https://gestorcreditos.com/faq\n- **Blog:** https://blog.gestorcreditos.com\n- **Centro de Ayuda:** https://help.gestorcreditos.com\n\n## 📋 Checklist de Instalación\n\n- [ ] Dispositivo Android 6.0 o superior\n- [ ] ADB instalado (si usas método 1)\n- [ ] Depuración USB habilitada\n- [ ] Conexión a Internet activa\n- [ ] Espacio libre en dispositivo (100+ MB)\n- [ ] APK descargado\n- [ ] Aplicación instalada\n- [ ] Cuenta creada\n- [ ] Workspace configurado\n- [ ] Notificaciones habilitadas\n\n## 🎉 ¡Listo!\n\nTu aplicación Gestor de Créditos está instalada y lista para usar.\n\n**Próximos pasos:**\n1.
