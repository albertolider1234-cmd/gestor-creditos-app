# ⚡ Termux - Inicio Rápido\n\n## 🎯 Compilar APKs Directamente en Android\n\n### Requisitos\n- Android 8.0+\n- 4 GB RAM mínimo\n- 10 GB espacio libre\n- Termux instalado\n\n### 5 Pasos Rápidos\n\n#### 1️⃣ Instalar Termux (5 min)\n```\nGoogle Play Store → Busca \"Termux\" → Instala\n```\n\n#### 2️⃣ Actualizar (10 min)\n```bash\npkg update && pkg upgrade -y\n```\n\n#### 3️⃣ Instalar Dependencias (15 min)\n```bash\npkg install -y nodejs npm git\n```\n\n#### 4️⃣ Obtener Proyecto (5 min)\n```bash\ncd $HOME\ngit clone https://github.com/tu-usuario/gestor-creditos-app.git\ncd gestor-creditos-app\n```\n\n#### 5️⃣ Compilar (30-40 min)\n```bash\nchmod +x compile-termux.sh\n./compile-termux.sh\n```\n\n### ✅ ¡Listo!\n\nLos APKs están en: `dist/apks/`\n\n### Instalar\n```bash\nadb install dist/apks/gestor-creditos-admin-*.apk\nadb install dist/apks/gestor-creditos-guest-*.apk\n```\n\n### Comandos Útiles\n\n```bash\n# Solo Admin\n./compile-termux.sh admin\n\n# Solo Invitado\n./compile-termux.sh guest\n\n# Limpiar\n./compile-termux.sh clean\n\n# Ayuda\n./compile-termux.sh help\n```\n\n### Solución Rápida\n\n| Problema | Solución |\n|----------|----------|\n| \"Comando no encontrado\" | `chmod +x compile-termux.sh` |\n| \"Node.js no encontrado\" | `pkg install -y nodejs npm` |\n| \"npm install falló\" | `npm cache clean --force` |\n| \"Espacio insuficiente\" | `pkg clean` |\n\n### ⏱️ Tiempo Total: 70-80 minutos\n\n---\n\nPara más detalles, lee: **TERMUX_BUILD_GUIDE.md**\n
